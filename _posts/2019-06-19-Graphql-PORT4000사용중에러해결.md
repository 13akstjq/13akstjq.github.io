> ### 포트 죽이는 방법

* 포트번호확인하기 
```
netstat -a -o
```

나온 결과에서 사용중이라는 포트번호에 속하는 PID값을 아래의 command에 입력하면 

```
taskkill /f /pid [PIDNUMBER]
```
