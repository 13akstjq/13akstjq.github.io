* pakage.json 에서 변경이 가능합니다. 
  ```
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "dev": "nodemon --exec babel-node src/server.js"
  },
  ```
  아래 명령어로 실행이 가능합니다. 
  - $npm run-script dev
