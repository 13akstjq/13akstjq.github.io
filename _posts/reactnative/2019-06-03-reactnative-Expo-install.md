> ## Expo 설치 
RN의 능력을 더욱 빛나게 해줄 Expo를 설치하려고 한다. Expo는 Reactnative로 수정한 프로젝트를 hot-code로 적용시킬 수 있다. 

콘솔에 다음과 같이 입력한다. 
```
$ npm install -g expo-cli
```

전에 만든 App을 build하는 방법
```
$ expo start
$ expo start --android
```
